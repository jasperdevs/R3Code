## Status Key

- `ported`: implemented in Rust/GPUI and verified against upstream by tests, screenshot parity, or a direct runtime proof.
- `partial`: some upstream contracts or visual states exist, but the surface is not complete.
- `missing`: no meaningful Rust implementation exists yet.
- `different`: intentionally different, with documented approval.
